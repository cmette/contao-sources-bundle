# contao-sources-bundle
Dieses Bundle bietet eine einfache Verwaltung von Quellen für das CMS Contao. 
Quellen wird hier in einem **weiten Sinne** verwendet und meint vor allem **literarische Quellen, Karten, Pläne und Risse, aber auch Fotos, Websites** und andere Entitäten, auf die in Texten oder von Websites aus verweisen werden soll.  

Zurzeit befindet es sich noch in Entwicklung. 

Folgende Funktionen sind implementiert  

1. Quellen können erfasst und verwaltet werden  
2. Quellen können typisiert werden, also jede Quelle kann einem Typ zugeordnet werden. Die Typen können vom Anwender/der Anwenderin selbst festgelegt werden (z.b. Monografie, Sammelband, Aufsatz, Reihe, Landkarte, Foto etc.)
3. Autorinnen und Autoren können unabhängig verwaltet werden
4. neue Inserttags wie Zitat:{{quote}} und Quelle:{{source}} sind implementiert, weitere sollen hinzukommen
4. [geplant] Reihen und Zeitschriften können separat verwaltet werden
